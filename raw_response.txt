## SHORT NOTES

### Detailed Definition with Examples

Dijkstra's Algorithm is a graph search algorithm that finds the shortest path between two nodes in a weighted graph. The algorithm works by maintaining a list of visited nodes and their shortest distances from the source node. It iteratively selects the node with the minimum distance and updates the distances of its neighboring nodes. The algorithm can be applied to both directed and undirected graphs.

Example: Suppose we have a weighted graph with nodes A, B, C, and D, and edges AB, BC, CD, and DA. If we want to find the shortest path from node A to node D, we can use Dijkstra's Algorithm.

### Types and Classifications

Dijkstra's Algorithm can be classified into two types:

1. **Single-Source Shortest Path (SSSP)**: This algorithm finds the shortest path from a single source node to all other nodes in the graph.
2. **All-Pairs Shortest Path (APSP)**: This algorithm finds the shortest path between all pairs of nodes in the graph.

### Step by Step Working Principle

The working principle of Dijkstra's Algorithm can be summarized as follows:

1. Initialize the distance to the source node as 0, and the distance to all other nodes as infinity.
2. Create a list of unvisited nodes and initialize the priority queue with the source node.
3. While the priority queue is not empty, extract the node with the minimum distance from the queue.
4. For each neighboring node of the extracted node, calculate the tentative distance and update the distance if it is less than the current distance.
5. Mark the extracted node as visited and remove it from the list of unvisited nodes.
6. Repeat steps 3-5 until the destination node is reached.

### C Implementation

```c
#include <stdio.h>
#include <stdlib.h>
#include <limits.h>

#define INF INT_MAX

typedef struct Node {
    int distance;
    int vertex;
} Node;

typedef struct Graph {
    int numVertices;
    int** adjMatrix;
} Graph;

void dijkstra(Graph* graph, int source) {
    int* distances = (int*)malloc(graph->numVertices * sizeof(int));
    int* visited = (int*)malloc(graph->numVertices * sizeof(int));

    for (int i = 0; i < graph->numVertices; i++) {
        distances[i] = INF;
        visited[i] = 0;
    }

    distances[source] = 0;

    for (int i = 0; i < graph->numVertices - 1; i++) {
        int minVertex = -1;
        for (int j = 0; j < graph->numVertices; j++) {
            if (!visited[j] && (minVertex == -1 || distances[j] < distances[minVertex])) {
                minVertex = j;
            }
        }

        visited[minVertex] = 1;

        for (int k = 0; k < graph->numVertices; k++) {
            if (!visited[k] && graph->adjMatrix[minVertex][k] != 0) {
                int newDistance = distances[minVertex] + graph->adjMatrix[minVertex][k];
                if (newDistance < distances[k]) {
                    distances[k] = newDistance;
                }
            }
        }
    }

    printf("Shortest distances from node %d:\n", source);
    for (int i = 0; i < graph->numVertices; i++) {
        printf("Node %d: %d\n", i, distances[i]);
    }
}

int main() {
    Graph graph;
    graph.numVertices = 5;
    graph.adjMatrix = (int**)malloc(graph.numVertices * sizeof(int*));
    for (int i = 0; i < graph.numVertices; i++) {
        graph.adjMatrix[i] = (int*)malloc(graph.numVertices * sizeof(int));
    }

    graph.adjMatrix[0][1] = 4;
    graph.adjMatrix[0][2] = 2;
    graph.adjMatrix[1][3] = 5;
    graph.adjMatrix[2][3] = 1;
    graph.adjMatrix[3][4] = 3;

    dijkstra(&graph, 0);

    return 0;
}
```

### Python Implementation

```python
import sys
import heapq

def dijkstra(graph, source):
    distances = {node: sys.maxsize for node in graph}
    distances[source] = 0

    pq = [(0, source)]

    while pq:
        current_distance, current_node = heapq.heappop(pq)

        if current_distance > distances[current_node]:
            continue

        for neighbor, weight in graph[current_node].items():
            distance = current_distance + weight

            if distance < distances[neighbor]:
                distances[neighbor] = distance
                heapq.heappush(pq, (distance, neighbor))

    return distances

graph = {
    0: {1: 4, 2: 2},
    1: {3: 5},
    2: {3: 1},
    3: {4: 3}
}

distances = dijkstra(graph, 0)
print(distances)
```
